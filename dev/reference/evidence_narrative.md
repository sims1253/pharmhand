# Evidence Narrative Generation

Functions for automatic generation of evidence narratives from
structured meta-analysis results, supporting IQWiG/G-BA style reporting.

## References

IQWiG Allgemeine Methoden Version 3.1.4, Section 3.1, p.51-56.
<https://www.iqwig.de/methoden/allgemeine-methoden-version-3-1.pdf>
