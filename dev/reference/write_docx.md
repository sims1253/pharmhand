# Write clinical content to a Word document

Generic function for writing clinical content to a Word document at a
specified path.

## Usage

``` r
write_docx(x, path, ...)
```

## Arguments

- x:

  A ClinicalReport, StudyResult, or ClinicalContent object

- path:

  Character string specifying the output file path

- ...:

  Additional arguments passed to methods

## Value

The modified object (invisibly)
