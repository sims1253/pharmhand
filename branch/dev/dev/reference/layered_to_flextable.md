# Convert LayeredTable to flextable

Build and style a LayeredTable as a flextable.

## Usage

``` r
layered_to_flextable(x, style = "clinical", ...)
```

## Arguments

- x:

  LayeredTable object

- style:

  Style preset (default: "clinical")

- ...:

  Additional arguments passed to apply_clinical_style

## Value

A styled flextable object
