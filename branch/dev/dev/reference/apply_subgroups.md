# Apply Subgroup Analysis

Apply Subgroup Analysis

## Usage

``` r
apply_subgroups(data, subgroup_var, analysis_fn, ...)
```

## Arguments

- data:

  ADaMData object or data frame

- subgroup_var:

  Character string specifying the subgroup variable

- analysis_fn:

  Function to perform analysis

- ...:

  Additional arguments

## Value

A list of AnalysisResults per subgroup
