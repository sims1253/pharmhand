# Write ClinicalReport to Word (S7 Method)

Write ClinicalReport to Word (S7 Method)

## Usage

``` r
write_docx_ClinicalReport(x, path, add_toc = FALSE, ...)
```

## Arguments

- x:

  ClinicalReport object

- path:

  File path

- add_toc:

  Logical, whether to add table of contents (default: FALSE)

- ...:

  Additional arguments
