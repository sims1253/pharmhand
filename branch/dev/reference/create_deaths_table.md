# Create Deaths Summary Table

Create Deaths Summary Table

## Usage

``` r
create_deaths_table(
  adsl,
  title = "Deaths Summary",
  trt_var = "TRT01A",
  autofit = TRUE
)
```

## Arguments

- adsl:

  ADSL data frame

- title:

  Table title

- trt_var:

  Treatment variable name

## Value

ClinicalTable object
