# Format Confidence Interval

Format Confidence Interval

## Usage

``` r
.format_ci(ci, decimals = 2, language = "en")
```

## Arguments

- ci:

  Numeric vector of length 2: c(lower, upper).

- decimals:

  Integer. Number of decimal places.

- language:

  Character. Language: "en" or "de".

## Value

Formatted CI string.
