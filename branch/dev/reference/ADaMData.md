# ADaMData Class

Base class for ADaM datasets with population filters.

## Usage

``` r
ADaMData(
  data = structure(list(), names = character(0), row.names = integer(0), class =
    "data.frame"),
  domain = "",
  population = "FAS",
  subject_var = "USUBJID",
  trt_var = "TRT01P",
  metadata = list()
)
```
