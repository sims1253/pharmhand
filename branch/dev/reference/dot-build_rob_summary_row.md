# Build RoB Summary Row

Internal helper to build a single row of data for the RoB summary table.

## Usage

``` r
.build_rob_summary_row(rob_result, include_justification)
```

## Arguments

- rob_result:

  RoB2Result object containing risk of bias assessment

- include_justification:

  Logical, include justification column

## Value

List of row data with all column values
