# Calculate Proportion Confidence Interval

Calculate Proportion Confidence Interval

## Usage

``` r
calculate_proportion_ci(x, n, method = "wilson", conf_level = 0.95)
```

## Arguments

- x:

  Number of successes

- n:

  Total number of trials

- method:

  CI method: "wilson", "exact", or "wald"

- conf_level:

  Confidence level

## Value

List with lower and upper bounds
