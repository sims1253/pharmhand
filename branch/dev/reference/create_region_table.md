# Create Enrollment by Region Table

Create Enrollment by Region Table

## Usage

``` r
create_region_table(
  adsl,
  title = "Enrollment by Region",
  trt_var = "TRT01P",
  region_var = "REGION1",
  autofit = TRUE
)
```

## Arguments

- adsl:

  ADSL data frame

- title:

  Table title

- trt_var:

  Treatment variable name

- region_var:

  Region variable name

## Value

ClinicalTable object
