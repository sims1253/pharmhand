# Extract MMRM Model Summary

Extracts summary statistics from an MMRMResult object.

## Usage

``` r
summary_mmrm(result, digits = 3)
```

## Arguments

- result:

  An MMRMResult object

- digits:

  Integer. Number of decimal places to round to

## Value

A data frame with model summary statistics

## Examples

``` r
if (FALSE) { # \dontrun{
summary <- summary_mmrm(result)
print(summary)
} # }
```
