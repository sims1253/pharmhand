# Plot Cumulative Incidence Function

Creates a plot of the cumulative incidence function from competing risk
analysis.

## Usage

``` r
plot_cif(result, title = "Cumulative Incidence Function")
```

## Arguments

- result:

  A CompetingRiskResult object

- title:

  Plot title

## Value

A ClinicalPlot object

## Examples

``` r
if (FALSE) { # \dontrun{
plot <- plot_cif(result, title = "Cumulative Incidence Function")
} # }
```
