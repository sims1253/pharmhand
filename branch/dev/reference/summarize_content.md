# Generate a summary of clinical content

Generic function for generating summary information about clinical
content objects.

## Usage

``` r
summarize_content(x, ...)
```

## Arguments

- x:

  A clinical content object

- ...:

  Additional arguments passed to methods

## Value

A list with summary information

## Examples

``` r
if (FALSE) { # \dontrun{
summary_info <- summarize_content(clinical_table)
} # }
```
