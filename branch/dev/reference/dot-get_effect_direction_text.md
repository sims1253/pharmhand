# Get Effect Direction Text

Get Effect Direction Text

## Usage

``` r
.get_effect_direction_text(
  estimate,
  effect_measure,
  direction,
  language = "en"
)
```

## Arguments

- estimate:

  Numeric effect estimate.

- effect_measure:

  Character effect measure type.

- direction:

  Character. "benefit", "harm", or "none".

- language:

  Character language.

## Value

List with direction text and adjective.
